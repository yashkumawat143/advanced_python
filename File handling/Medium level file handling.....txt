Medium level file handling....



1....Count the total number of lines in a file.
answer :

count=0;
with open("student.txt","r") as file:
    for i in file:
        count+=1;
print("Total line in file : ",count);




2....Count the total number of words.
answer :

count=0;
with open("student.txt","r") as file:
    for i in file:
        ans=i.split();
        count+=len(ans);
print("Total words in file : ",count);
