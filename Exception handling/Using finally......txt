1....Open a file.
Print its contents.
Whether the file exists or not, print:

answer :

try:
    n1=input("Enter the file name : ");
    f=open("n1","r");
    print(f.read());
except FileNotFoundError:
    print("File not found");
finally:
    print("The file is close");
